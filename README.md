# glamrous.me [![Build Status](https://travis-ci.org/DrewRomanyk/glamrous.me.svg?branch=master)](https://travis-ci.org/DrewRomanyk/glamrous.me)
---

Setup:

```bash
; cd dockerfile
; docker build -t glamrous-dev .
; docker build -t glamrous-server .
```

Run server:

```bash
; ./start.sh
```

Build for frontend:

```bash
; ./build.sh
```
